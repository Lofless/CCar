## 二、项目简介

本项目基于 MuJoCo 物理仿真引擎与模型预测控制（MPC）框架，构建了一个包含车辆运动控制与实时仪表盘显示的仿真系统。系统通过对车辆位姿、速度及控制输入进行建模，实现了车辆向目标点自主运动的控制过程。同时，在仿真界面中设计并绘制了简化的车辆仪表盘，用于动态展示当前行驶状态，如速度变化和方向信息。该项目重点体现了 MPC 在连续控制问题中的应用方式，以及仿真系统中任务逻辑、可视化模块与控制算法之间的协同关系，具有一定的教学和实验价值。
